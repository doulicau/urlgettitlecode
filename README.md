可以批量多线程测试url的状态码、title、跳转后的页面，结果保存到:执行脚本时间.xls中。  

测试1W个站点，需要时间不到10分钟。  

使用方法：

测试的url放到url.txt中，然后：

python3 gettitlecode.py

结果输出到:执行脚本时间.xls中
